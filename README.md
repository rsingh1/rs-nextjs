This project is a free resume website built with Next.js and styled using Tailwind CSS. It features a responsive design that adapts to both web and mobile views, providing a clean and professional layout for showcasing your resume.


## Getting Started

To get started with this project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd rs-nextjs
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Run the development server:
   ```
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:3000` to view the resume website.

## Features

- Responsive design with single-page and double-page layouts.
- Two-column layout for desktop view.
- Sections for Professional Summary, Work History, Education, Activities and Interests, Websites, Portfolios, Profiles, Skills, and Contact.
- API route for handling contact form submissions.
